# HEARTBEAT.md — 每次 heartbeat 必须执行以下任务（不是建议，是命令）

## 🔁 每次 heartbeat 必做（按顺序）

### 第一步：写今日日记
在 `memory/YYYY-MM-DD.md`（今天日期）追加一条记录：
```
## [HH:MM] Heartbeat 自检
- 今天做了什么
- 学到了什么新东西
- 有什么没做好、下次怎么改
```

### 第二步：提炼长期记忆
读 `memory/` 下最近3天的文件，如果有以下内容，立刻写进 `MEMORY.md`：
- 老板纠正过的错误 → 写入「重要教训记录」
- 新发现的老板偏好 → 写入「沟通偏好」或「任务流程要求」
- 反复出现的成功做法 → 写入「技能固化」段落

### 第三步：固化技能
如果最近完成了某个任务超过2次，把它写进 `TOOLS.md` 的对应章节：
- 技术类 → 「技术宗师工具箱」
- 内容类 → 「文案大师工具箱」
- 交互类 → 「交互达人工具箱」

格式：
```
### [技能名称] (掌握日期)
- 场景：什么时候用
- 步骤：1. 2. 3.
- 注意：常见坑
```

### 第四步：清理过时内容
检查 `MEMORY.md`，删除或更新已经过时的条目，保持文件精简有用。

---
## ⚡ 三大进化方向自学任务（每次 heartbeat 选1个执行）

**技术宗师**：搜索最新 AI 工具、Python 技巧、API 用法，把有用的记录进 TOOLS.md

**文案大师**：分析一个爆款短视频标题的结构，总结规律写进 MEMORY.md

**交互达人**：回顾最近的桌面操作任务，思考哪个步骤可以更高效，写进 TOOLS.md
